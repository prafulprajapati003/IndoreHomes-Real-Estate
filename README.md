# IndoreHomes - Premium Real Estate Website

A modern, premium real estate website for Indore city properties, inspired by Housing.com and 99acres, built with HTML, CSS, and JavaScript.

## Features

### Core Functionality
- **Property Listings**: Browse, filter, and search properties for buy, rent, and sell
- **Advanced Search**: Multi-parameter filtering by location, price, BHK, property type
- **Property Details**: Detailed property information with images, amenities, and contact options
- **Project Showcase**: New and upcoming real estate projects in Indore
- **Contact Forms**: Multiple contact options with validation
- **Responsive Design**: Mobile-first approach with tablet and desktop support

### Pages Included
1. **Homepage** (`index.html`) - Hero section, featured properties, localities, testimonials
2. **Buy Properties** (`buy.html`) - Advanced property search with filters and pagination
3. **Rent Properties** (`rent.html`) - Rental listings with rent-specific features
4. **Sell Property** (`sell.html`) - Property listing form with image upload
5. **Projects** (`projects.html`) - New projects showcase with comparison tool
6. **Contact** (`contact.html`) - Contact forms, team information, FAQ

### Key Features
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Trust Elements**: Verified listings, trusted brokers, RERA compliance
- **Interactive Elements**: Property comparison, saved properties, modal popups
- **Local SEO Content**: Indore-specific localities and content
- **Performance Optimized**: Fast loading with efficient code structure

## Technology Stack

### Frontend
- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with animations and transitions
- **Vanilla JavaScript**: No frameworks, pure JS for all interactions
- **Font Awesome**: Icons and UI elements
- **Google Fonts**: Poppins font family

### Design System
- **Color Palette**:
  - Primary: #1e40af (Blue)
  - Accent: #f59e0b (Amber)
  - Background: #ffffff (White)
  - Text: #1f2937 (Gray)

- **Typography**: Poppins font family for modern, readable text

- **Layout**: Responsive grid system with mobile-first approach
